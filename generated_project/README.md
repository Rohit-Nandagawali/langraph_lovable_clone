# SimpleCalculator

## Brief Description
SimpleCalculator is a lightweight web-based calculator that performs basic arithmetic operations. It provides a clean and responsive user interface built with HTML, CSS, and JavaScript, allowing users to quickly compute addition, subtraction, multiplication, and division directly in the browser.

---

## Tech Stack
- **HTML** – Structure of the calculator UI.
- **CSS** – Styling and responsive layout.
- **JavaScript** – Core logic for handling input, performing calculations, and updating the display.

---

## Setup Instructions
1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd <repository-directory>
   ```
2. **Open the application**
   - Locate the `index.html` file in the project root.
   - Open `index.html` in any modern web browser (Chrome, Firefox, Edge, Safari, etc.).
   - No additional server or build steps are required.

---

## Usage Guide
- **Input Fields**: Enter numbers into the two input boxes labeled *Number 1* and *Number 2*.
- **Operation Buttons**:
  - **Add (+)** – Calculates the sum of the two numbers.
  - **Subtract (−)** – Calculates the difference (Number 1 − Number 2).
  - **Multiply (×)** – Calculates the product of the two numbers.
  - **Divide (÷)** – Calculates the quotient (Number 1 ÷ Number 2). Division by zero is handled gracefully with an error message.
- **Result Display**: The result of the selected operation appears below the buttons.
- **Clear Button**: Click **Clear** to reset both input fields and the result display.
- **Responsive Behavior**: The layout adjusts automatically for different screen sizes, ensuring usability on desktop, tablet, and mobile devices.

---

## Core Implementation
- **`index.html`** – Contains the markup for the calculator UI.
- **`style.css`** – Provides styling, including responsive design rules.
- **`script.js`** – Implements the calculation logic, event handling, and UI updates.
